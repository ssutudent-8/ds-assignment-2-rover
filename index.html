■ 클래스 설명

RobotADT

탐사 / 충전 / 상태조회의 추상 인터페이스. 모든 로봇의 공통 계약을 정의합니다.

BaseRobot

배터리, 효율, 누적 탐사량, 대기턴을 관리하는 구체 베이스 클래스. _set_battery로 범위를 보정합니다.

QuadrupedRobot

Night 충전 불가. Day에 recharge(weather) 호출 시 즉시 완전충전. 대기 없음.

WheeledRobot

즉시 완전충전. 충전 후 2턴 대기(탐사 불가). 날씨 무관하게 충전 가능.

World

날씨(낮/밤), 총 실행 횟수, 턴을 관리. 설정된 주기마다 낮↔밤 자동 전환.
